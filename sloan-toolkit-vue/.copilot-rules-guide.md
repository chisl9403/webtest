# Copilot 自动审批配置说明

## 已启用的自动审批规则

### 1. 配置文件修改 ✅
自动审批以下文件的修改：
- `config.json`, `config.local.json`
- `*.config.js`, `*.config.ts`
- `tsconfig.json`, `package.json`

### 2. 样式调整 ✅
自动审批：
- SCSS/CSS 文件修改
- Vue 组件的 style 部分

### 3. 布局调整 ✅
自动审批以下操作：
- 调整间距、宽度、高度
- 元素对齐
- 字体大小、颜色修改

### 4. 新增功能模块 ✅
自动审批：
- 添加新卡片组件
- 新增数据源和 API 调用
- 添加新组件

### 5. 代码重构 ✅
自动审批：
- 提取函数
- 重命名变量
- 优化代码结构
- 添加注释

## 需要确认的操作 ⚠️

以下操作仍需要用户确认：
- 删除函数
- 修改 API 密钥
- 删除数据库
- 修改认证逻辑

## 代码生成设置

- ✅ 自动格式化代码
- ✅ 自动添加注释
- ✅ 遵循项目规范
- 单引号、无分号、2空格缩进

## 通知设置

- 简短确认模式
- 仅显示成功消息和错误
- 不显示详细日志

## 使用方法

1. 现在进行配置和代码调整时会自动执行
2. 只在完成后显示简短确认
3. 如需修改规则，编辑 `.copilot-rules.json`
